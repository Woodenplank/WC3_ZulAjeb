# WC3_ZulAjeb

### External dependencies
These libraries, by other authors, are needed for at least one of the triggers to function properly.

* [Total Initialization](https://www.hiveworkshop.com/threads/total-initialization.317099/) by Bribe of Hiveworkshop.com
* [Relativistic Missiles](https://www.hiveworkshop.com/threads/relativistic-missiles-vjass-lua-gui.325964/) by Chopinski of Hiveworkshop.com
    * For these two to work in tandem, a slight modification must be made to the latter. Since Chopinski included his own "OnInit()" function, which conflicts with Bribe's version.
    Therefore I'm using a slightly edited version of Chopinski's system, with the initialization segment replaced.
